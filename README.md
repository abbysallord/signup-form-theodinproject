# Signup Form - The Odin Project

A signup form project built as part of The Odin Project curriculum.

## Description

This project demonstrates form validation, HTML structure, and CSS styling techniques. It includes client-side validation and responsive design principles.

## Features

- User input validation
- Form styling with CSS
- Responsive layout
- Error message handling

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Test the form with various inputs

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Learning Outcomes

- Form validation techniques
- DOM manipulation
- CSS layout and styling
- User experience best practices
